        ResultGatherer

This class listens on a UDP port for kernels reporting stdout and stderr.

The current implementation simply prints the stdin, stdout and stderr of each kernel command to standard out in colorized form.

The IPython.ColorANSI module is used to colorize the output.

Currently I use diconnected UDP. There could be a benefit in going over to connected mode.

Also, currently, I don't report the port that the kernel is listening on. This information will be useful when multiple kernel's are running on a single interface.
