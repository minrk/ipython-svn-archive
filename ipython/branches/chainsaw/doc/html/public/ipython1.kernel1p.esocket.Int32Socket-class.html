Type Int32Socket

object --+
         |
        Int32Socket

A wrapper for a socket that reads/writes int32 prefixed strings.

This sends and receives strings over a socket, prefixing them with a 4 byte integer specifying the length of the string. This 4 byte integer is encoded in network byte order.

Notes

  1. If extra data is received by this class it is discarded.
  2. This class might work with blocking/timeout'd sockets.
  3. No socket related exceptions are caught.
